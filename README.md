# ModuleRobot
Projet Tutoré IUT 2014-2015

##Description
Module situé dans chaque robot Lego NXT.

Communique avec le module RaspberryRobot, pour recevoir les ordres de déplacements.

##Installation
Doit être compilé comme *LejosNXT Program*.

Ne pas oublier de connecter le robot en USB, d'avoir installé LejosNXT et les drivers sur l'ordinateur de développement.
